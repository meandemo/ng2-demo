Dynamic Slider

We show an example a dynamic slider: runners can be added or removed. It is a customization of the simple svg slider. We have created a new html tag <gg-svg-slider-dyn> which can be used like the standard <gg-svg-slider> slider. Note that a click on the rail has no effect.

Communication With Slider Component

You may note that we not using a bi-directional binding for the values. The slider emit an array of values, which is capture in the parent component with the syntax (values)="_values". Notice, however, that the we can move one of the sliders using an number input field in the control panel.

HTML Code

        <gg-svg-slider-dyn 
          (values)="_values = $event"
          [length]="my_rail_length_">
        </gg-svg-slider-dyn>
