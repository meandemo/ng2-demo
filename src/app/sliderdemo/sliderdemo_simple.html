Standard Slider

The slider below is a standard html5 slider using the <input type="range" ... > html tag. It's a good example of bi-directional binding with the special angular syntax: [(ngModel)]: the slider and value input field are linked. The three input items: value, min and max are grouped in a html <form> tag which allows the activation of powerful check functions. In our case, we check that the min value is always less than max and that the value is changed when either min or max are updated.

To illustrate these checkers, you can try to change the min value to be greater than the current value. This change is rejected and the min value remains unchanged. This is done with the special attribute ngFormModel and used with the form tag as follows: <form [ngFormModel]="my_ctrl">. The variable my_ctrl has type ControlGroup and it must be a member variable of your component class.

SVG Based Slider

We propose to use svg to implement a fully custom slider without the clever css tricks commonly found to customize the html default slider. The figure below describes an svg slider with its bounding box, the rail and the runner.

The slider shown just below is a custom slider implemented with the angular2 framework. We have a new html tag <gg-svg-slider> which can be used like the standard <input> slider.
